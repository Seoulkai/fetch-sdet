# fetch-sdet
Take home test for Fetch SDET position

Setup
-------------------------------------------------------------------------------
- Install Ruby (version 3.0.0 or greater)
- Install bundler
   - "gem install bundler"
- Run "bundle install"

This should install all the gems required to run the utility

Usage
-------------------------------------------------------------------------------
- "./geocodes.sh {city,state} {zipcode}"


Tests
-------------------------------------------------------------------------------
- Run "cucumber"
